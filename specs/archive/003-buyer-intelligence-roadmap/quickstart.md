# Validation Quickstart: Buyer Intelligence Roadmap

1. Run `uv run pytest -q tests/data_service tests/agent_api`.
2. Run `npm run test:gate` and `npm run build` from `frontend/`.
3. Submit a brief with a must-have and deal-breaker; verify the visible decision profile and factual gaps.
4. Compare homes and verify unit values, source/snapshot evidence, buyer-entered costs, and historical context or an honest unavailable state.
5. Save research, load a different result set, and verify added/removed/unchanged state.
6. Verify map area labels and overlap count with repeated coordinates.
