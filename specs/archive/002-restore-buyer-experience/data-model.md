# Data Model: Restore Buyer Experience

No persistent model changes.

## Presentation State

- **Landing**: Initial editorial state with no fabricated results.
- **Researching**: Existing trace/progress state after a buyer submits a brief.
- **Evidence result**: Existing active, historical, or partial property result set.
- **Decision state**: Existing comparison, decision-sheet, and selected-location states.

All states retain the existing source, snapshot, location, and comparison data contracts.
