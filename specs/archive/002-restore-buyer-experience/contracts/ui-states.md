# UI State Contract

| State | Required buyer-visible behavior |
|---|---|
| Initial | Editorial landing, active-dataset disclosure, primary brief action, no synthetic listing or confidence claim |
| Researching | Existing progress is visible without removing brief context |
| Results | Evidence, source, snapshot, map, and comparison remain available |
| Empty or partial | Clear unavailable language; no fabricated value or broken layout |
| Failure | Safe actionable message and retained recovery path |
