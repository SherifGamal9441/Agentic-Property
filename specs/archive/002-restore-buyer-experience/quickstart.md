# Validation Quickstart: Restore Buyer Experience

1. Run `npm run test:gate` from `frontend/`.
2. Run `npm run build` from `frontend/`.
3. Open the buyer UI at desktop and mobile widths.
4. Verify the screenshot traits: editorial hero, honest supporting panel, four-step journey, guided starts, and location-aware workspace.
5. Submit a buyer brief and verify source/snapshot evidence, map selection, one-to-four comparison, saved-search message, decision sheet, and failure state remain usable.
