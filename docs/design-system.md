# Design system

## Direction

Editorial Dubai carries reading, criteria editing, evidence, and decision work. Dark cinematic surfaces mark live intelligence, map evidence, and selected comparison. Abstract architectural geometry avoids fabricated listing imagery.

## Tokens

- Warm ivory `#f4f0e7` and paper `#fbf8f1`.
- Deep forest `#10231d` and secondary forest `#1d382f`.
- Champagne `#c7a66a`, terracotta `#a65f44`, ink `#17231f`.
- Serif display stack with a system sans-serif interface stack.

## Interaction rules

Motion communicates reveal, state transition, drawer movement, and hover depth only. Reduced motion is honored globally. Focus uses a visible warm outline. Controls target at least 44 px. Mobile retains primary navigation in a fixed dark tray. Missing values say “Not reported,” “Unknown,” or “Not verified”—never `0` or an empty decorative gap.

## Brand

The inline SVG/CSS Aizen monogram emphasizes `AI` with terracotta/champagne contrast and remains sharp without an image asset. Product branding is project-only.
