# Job-fair demo runbook

## Before doors open

1. Pull the exact DVC snapshot and run preflight.
2. Start Compose and wait for all four services to report healthy.
3. Warm the selected live model with a non-demo request.
4. Run all three presets once; do not save or replay their answers.
5. Use “Reset showcase” to clear browser-local decisions.

## Recruiter flow

Lead with the Dubai Marina preset. Pause on editable criteria to explain buyer confirmation. Expand the live trace and state that scoring and audit are deterministic. Open one evidence drawer, show an unknown, shortlist two homes, switch to area context, enter an affordability scenario, and open print preview. Finish at `#/case-study` and the architecture diagram.

## Failure recovery

If the provider fails, show the calm error as intentional live-only behavior, confirm preflight, and retry. If the map tiles fail, show the coordinate evidence fallback. Never switch to saved AI output.
